# TradeLog.dev 콘텐츠 플랜

## 사이트 정보
- **이름:** TradeLog.dev
- **도메인:** baksuls.com
- **주제:** AI 코인 자동매매 시스템 OWL 개발 일지 + 트레이딩 로그
- **언어:** 한국어/영어 동시
- **기술:** Astro + Cloudflare Pages

## 시리즈 4: "Daily Trading Log" (매일)

- 매일 1개: 오늘의 포지션, 진입/청산 내역, 수익률, 교훈
- 주간 1개: 주간 총정리, 전략별 성적, 조정사항

## 발행 일정

- **Phase 1 (1주차):** 시리즈1 #2~#5 → 하루 1개 (한/영 동시)
- **Phase 2 (2주차):** 시리즈1 #6~#10 + Daily Log 시작
- **Phase 3 (3주차~):** 시리즈2, 3 병행 + Daily 유지

## 글 톤 & 스타일

- 1인칭 서술 ("나는 ~했다")
- 실패를 숨기지 않는다
- 핵심 코드 스니펫 포함 (전체 코드 X)
- 데이터/백테스트 결과표 포함
- 각 글 끝에 "이 글에서 배운 것" 정리
