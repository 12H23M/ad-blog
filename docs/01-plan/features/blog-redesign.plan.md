## 1. 개요

### 1.1 목표
AdSense 승인에 적합한 전문적이고 깔끔한 블로그 디자인으로 리브랜딩

### 1.2 범위
- 블로그 이름 및 브랜딩 확정
- 로고 및 헤더 디자인 개선
- 전체 UI/UX 디자인 개선
- 모바일 반응형 최적화

---

## 2. 블로그 이름 후보

### 2.1 네이밍 기준
- 기억하기 쉬운 짧은 이름
- 생활/기술/재테크 주제를 포괄
- 도메인 사용 가능 (baksuls.com 기존 도메인 활용)
- 한국어 친화적

### 2.2 후보 목록

| 이름 | 의미 | 장점 |
|------|------|------|
| **박설스** | 기존 도메인 (baksuls.com) 활용 | 이미 도메인 보유, 독특함 |
| **알쓸정보** | 알아두면 쓸모있는 정보 | 직관적, 블로그 성격 표현 |
| **일상플러스** | 일상에 플러스가 되는 정보 | 긍정적 의미, 기억 용이 |
| **스마트라이프** | Smart + Life | 현대적, 기술/생활 포괄 |

### 2.3 권장안
**박설스 (Baksuls)** - 기존 도메인과 일치하며 독특한 브랜드 아이덴티티 구축 가능

---

## 3. 디자인 방향

### 3.1 디자인 원칙
1. **미니멀리즘**: 불필요한 요소 제거, 콘텐츠 중심
2. **가독성**: 충분한 여백, 적절한 폰트 크기
3. **일관성**: 통일된 색상 팔레트, 타이포그래피
4. **신뢰감**: 전문적이고 깔끔한 인상

### 3.2 색상 팔레트 (안)

```
Primary:   #2563EB (파란색 - 신뢰, 전문성)
Secondary: #10B981 (초록색 - 성장, 재테크)
Accent:    #F59E0B (주황색 - 생활, 따뜻함)
Text:      #1F2937 (다크 그레이)
Background:#F9FAFB (라이트 그레이)
```

### 3.3 타이포그래피
- 제목: Pretendard Bold / 24-32px
- 본문: Pretendard Regular / 16-18px
- 메타: Pretendard Light / 14px
